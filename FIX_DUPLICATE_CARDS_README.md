# إصلاح تكرار البطاقات في الصفحة الرئيسية

## المشكلة
البطاقات في الصفحة الرئيسية تظهر مرتين (مكررة)

## الحل

### الطريقة السريعة:
1. افتح Supabase Dashboard
2. اذهب إلى SQL Editor
3. انسخ محتوى ملف `CLEAN_DUPLICATE_CARDS.sql`
4. شغّله (Run)
5. ارجع للتطبيق وأعد تحميل الصفحة

### الطريقة الكاملة:
1. شغّل ملف `CREATE_HOME_CARDS_TABLE.sql` المحدث بالكامل
2. هذا سيحذف التكرارات ويعيد إدراج البيانات الصحيحة

## النتيجة المتوقعة:
- 4 بطاقات فقط (بدون تكرار)
- ترتيب صحيح: خريطة → عقارات → إضافة → ملف شخصي

## التحقق:
```sql
SELECT title, display_order FROM home_cards ORDER BY display_order;
```
يجب أن ترى:
1. وين اسكن
2. العقارات المتوفرة الآن
3. إضافة عقار جديد
4. ملفي الشخصي
